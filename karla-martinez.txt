    1. What is Semantic HTML? 
Sematic HTML the recommended markup for HTML. It describes clearly the purpose and the type of content that the tags contain. In other words it describes tags with meaning.
    2. What is HTML used for? 
HTML is the standard hypertext markup language for the web. Every website on the world wide web consists of an html file.
    3. What is an attribute and where do we put it? 
All HTML elements can have attributes. Attributes provide additional information about an element. Attributes are always in the start tag and usually come in a name/value pairs like name="value".
    4. What is the h1 tag used for? How many times should I use it on a page?
The h1 tag is used as header 1. It marks up the content that is in between the tags so that search engines can successfully find it. It should only be used once to label the most important content on the page.

    7. What is an id? 
The id attribute specifies a unique id for an HTML element.
    8. What elements can I add an id to?

The class is an attribute which specifies one or more class names or an HTML element.

    16. What is the difference between a section and a div
A section means that the content inside is grouped and should appear as a entry in an outline of a page. A div doesn't convey meaning. It is typically used to group elements to format them in CSS.
    17. What is CSS used for? 
It describes how HTML elements are to be displayed on screen, paper, or in other media. It can control the layout of multiple web pages all at once.
